Problem 47

Rational Function Asymptotes
Given
H(x) = (x^3 - 8)/(x^2 - 5x + 6)
Step 1 - Write the Rational Function and Factor
Given: $$ H(x)=\frac{x3 - 8}{x2 - 5*x + 6} $$ Factor numerator and denominator: $$\text{Numerator} = (x - 2)·(x^2 + 2x + 4)$$ $$\text{Denominator} = (x - 3)·(x - 2)$$
Step 2 - Domain Restrictions and Simplified Form
Set denominator = 0 to find forbidden x-values: x = 2 x = 3 Simplified function (after canceling common factors): $$ H(x) = (x^2 + 2x + 4)/(x - 3) $$
Step 3 - Vertical Asymptotes and Holes
Vertical asymptotes come from the simplified denominator = 0: x = 3 Any canceled restriction gives a hole: x = 2 (hole at (2, -12))
Step 4 - Horizontal or Oblique Asymptote
For the simplified function: degree(top) = 2, degree(bottom) = 1. Rule used: numerator degree is exactly 1 more, so there is a slant (oblique) asymptote from polynomial division. No horizontal asymptote. Oblique asymptote: y = x + 5

Exam Answer
① Factor and Domain Restrictions
Numerator factorization : (x - 2)·(x^2 + 2x + 4) Denominator factorization: (x - 3)·(x - 2) Domain restrictions (from original denominator): x = 2 x = 3
② Vertical Asymptote Test
Simplified function: (x^2 + 2x + 4)/(x - 3) Set simplified denominator = 0: x = 3 -> vertical asymptote
③ Hole Check (canceled factors)
Cancellations create removable discontinuities (holes): x = 2 (hole at (2, -12))
④ Horizontal / Oblique Asymptote
Degree(top) = 2, Degree(bottom) = 1 Rule: numerator degree is exactly 1 more, so there is a slant (oblique) asymptote from polynomial division. Horizontal asymptote: None Oblique asymptote: y = x + 5

Problem 58

Rational Application
Given
P(t) = 50(1 + 0.5t)/(2 + 0.01t)
Step 1 - Write the Model
Population model: $$ P(t) = 2500*(t + 2)/(t + 200) $$ where t is measured in months.
Step 2 - Part (a): Population at t = 0
Substitute t = 0: $$ P(0) = 25 $$ So the initial population is 25.
Step 3 - Part (b): Population after 5 months
Substitute t = 5: $$ P(5) = 3500/41 $$ So after 5 months, population is 3500/41.
Step 4 - Part (c): Horizontal Asymptote and Meaning
Because numerator and denominator are both degree 1, the horizontal asymptote is ratio of leading coefficients: $$ y = 2500 $$ Interpretation: the population approaches 2500 insects as time becomes very large.
