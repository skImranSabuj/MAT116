★ Key Notes for this Section
Exponential Function
$f(x) = a^x$, where $a > 0$, $a \neq 1$. Domain: $(-\infty,\infty)$; Range: $(0,\infty)$; Horizontal asymptote: $y=0$. If $a > 1$: increasing (growth). If $0 < a < 1$: decreasing (decay).
Key Transformation Rules
• $f(x)+k$: shift up/down $k$ units (HA shifts to $y=k$) • $f(x+h)$: shift left ($h>0$) or right ($h<0$) • $-f(x)$: reflect over $x$-axis; range becomes $(-\infty,0)$ • $f(-x)$: reflect over $y$-axis ($a^{-x} = (1/a)^x$) • $cf(x)$: vertical stretch/compression by factor $c$
One-to-One Property
$a^u = a^v \Leftrightarrow u = v$ (for $a > 0$, $a \neq 1$). This property is the key to solving exponential equations by writing both sides with the same base.

Problem 75

Exponential Equation
Given
Solve $\;3^{x^2-7} = 27^{2x}$.
Write both sides as powers of 3
$27^{2x} = (3^3)^{2x} = 3^{6x}$ So $3^{x^2-7} = 3^{6x} \Rightarrow x^2-7 = 6x$
Solve the quadratic
$x^2 - 6x - 7 = 0 \Rightarrow (x-7)(x+1) = 0$
Exam Answer
Write both sides as powers of 3
$27^{2x} = (3^3)^{2x} = 3^{6x}$ So $3^{x^2-7} = 3^{6x} \Rightarrow x^2-7 = 6x$
Solve the quadratic
$x^2 - 6x - 7 = 0 \Rightarrow (x-7)(x+1) = 0$
Final Answer
$x = 7$ or $x = -1$

Problem 77

Exponential Equation
Given
Solve $\;9^{2x} \cdot 27^{x^2} = 3^{-1}$.
Express all terms in base 3
$9^{2x} = 3^{4x}$, $\;27^{x^2} = 3^{3x^2}$ So $3^{3x^2+4x} = 3^{-1} \Rightarrow 3x^2+4x = -1$
Solve the quadratic
$3x^2+4x+1 = 0 \Rightarrow (3x+1)(x+1) = 0$
Exam Answer
Express all terms in base 3
$9^{2x} = 3^{4x}$, $\;27^{x^2} = 3^{3x^2}$ So $3^{3x^2+4x} = 3^{-1} \Rightarrow 3x^2+4x = -1$
Solve the quadratic
$3x^2+4x+1 = 0 \Rightarrow (3x+1)(x+1) = 0$
Final Answer
$x = -\dfrac{1}{3}$ or $x = -1$
