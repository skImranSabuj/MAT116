Problem 43

Exp_transform
Given
f(x)=2^x+1. Determine transformations, domain, range, and horizontal asymptote.
Transformation
Start from y=2^x and shift up by 1.
Characteristics
Domain: (-inf,inf) Range: (1,inf) Horizontal asymptote: y=1
Exam Answer
Domain (-inf,inf), range (1,inf), HA y=1.

Problem 75

Exp_equation
Given
Solve 3^(x^2-7)=27^(2x).
Use common base
27^(2x)=(3^3)^(2x)=3^(6x). So x^2-7=6x.
Solve quadratic
x^2-6x-7=0 -> (x-7)(x+1)=0
Exam Answer
x=7 or x=-1.

Problem 77

Exp_equation
Given
Solve 9^(2x)*27^(x^2)=3^(-1).
Use base 3
9^(2x)=3^(4x), 27^(x^2)=3^(3x^2). Then 3^(3x^2+4x)=3^(-1).
Solve quadratic
3x^2+4x+1=0 -> (3x+1)(x+1)=0
Exam Answer
x=-1 or x=-1/3.

Problem 78

Exp_equation
Given
Solve 4^x*2^(x^2)=16^2.
Use base 2
4^x=2^(2x), 16^2=2^8. So 2^(x^2+2x)=2^8.
Solve quadratic
x^2+2x-8=0 -> (x+4)(x-2)=0
Exam Answer
x=2 or x=-4.

Problem 82

Exp_equation
Given
Solve (e^4)^(2x)*e^x=e^12.
Combine exponents
(e^4)^(2x)=e^(8x), so e^(8x)e^x=e^(9x)=e^12.
Solve
9x=12 -> x=4/3
Exam Answer
x=4/3.
