Given
Population model: P(t)=287(1.010)^(t-1999). Find when population reaches 307 million and 394 million.
Step 1 - Write the Exponential Model
Model: $$ 287·1.01^(t - 1999) $$ This is an exponential growth model.
Step 2 - Set Model Equal to Each Target Value
(a) Solve 287·1.01^(t - 1999) = 307 t = 100.49917080713·log(307/287) + 1999 ~= 2005.770 (b) Solve 287·1.01^(t - 1999) = 394 t = 100.49917080713·log(394/287) + 1999 ~= 2030.845
Step 3 - Solve Using Logarithms
Isolate the exponential expression, then take natural logs to solve for the time variable.
Step 4 - Interpret the Result
Interpret each solution in context (population) with units: million people. (a) t ~= 2005.770 (b) t ~= 2030.845
Exam Answer
(a) Target 307
Set model equal to target and solve: 287·1.01^(t - 1999) = 307 t = 100.49917080713·log(307/287) + 1999 ~= 2005.770 Approximate year/value marker: 2006
(b) Target 394
Set model equal to target and solve: 287·1.01^(t - 1999) = 394 t = 100.49917080713·log(394/287) + 1999 ~= 2030.845 Approximate year/value marker: 2031
