★ Key Notes for Section 7.2
Ranges of Inverse Trig Functions
$\sin^{-1}x$: range $\bigl[-\tfrac{\pi}{2}, \tfrac{\pi}{2}\bigr]$ (QI and QIV) $\cos^{-1}x$: range $[0, \pi]$ (QI and QII) $\tan^{-1}x$: range $\bigl(-\tfrac{\pi}{2}, \tfrac{\pi}{2}\bigr)$ (QI and QIV)
Strategy for Composite Problems
1. Evaluate the inner inverse trig function to get an angle $\theta$ in the correct range. 2. Identify which quadrant $\theta$ falls in (sign from the range restriction). 3. Evaluate the outer trig function at $\theta$, using exact values from the unit circle or a reference right triangle.
Key Exact Values (Unit Circle)
$\sin(\pi/6)=\tfrac{1}{2}$, $\cos(\pi/6)=\tfrac{\sqrt3}{2}$, $\tan(\pi/6)=\tfrac{\sqrt3}{3}$ $\sin(\pi/4)=\tfrac{\sqrt2}{2}$, $\cos(\pi/4)=\tfrac{\sqrt2}{2}$, $\tan(\pi/4)=1$ $\sin(\pi/3)=\tfrac{\sqrt3}{2}$, $\cos(\pi/3)=\tfrac{1}{2}$, $\tan(\pi/3)=\sqrt3$ $\sin(5\pi/6)=\tfrac{1}{2}$, $\cos(5\pi/6)=-\tfrac{\sqrt3}{2}$, $\tan(5\pi/6)=-\tfrac{\sqrt3}{3}$

Problem 9

Evaluate Composite Inverse Trig
Find the exact value
$$\cos\!\left(\sin^{-1}\frac{\sqrt2}{2}\right)$$
Step 1 — Evaluate the inner function
Let $\theta = \sin^{-1}\!\dfrac{\sqrt2}{2}$. Since $\sin(\pi/4)=\dfrac{\sqrt2}{2}$ and $\theta \in [-\pi/2, \pi/2]$: $$\theta = \frac{\pi}{4}$$
Step 2 — Evaluate the outer function
$$\cos\!\left(\frac{\pi}{4}\right) = \frac{\sqrt2}{2}$$
Final Answer
$$\cos\!\left(\sin^{-1}\frac{\sqrt2}{2}\right) = \frac{\sqrt2}{2}$$

Problem 10

Evaluate Composite Inverse Trig
Find the exact value
$$\sin\!\left(\cos^{-1}\frac{1}{2}\right)$$
Step 1 — Inner function
$\cos(\pi/3)=\tfrac{1}{2}$, and $\pi/3 \in [0,\pi]$, so $\cos^{-1}\!\tfrac{1}{2}=\dfrac{\pi}{3}$.
Step 2 — Outer function
$$\sin\!\left(\frac{\pi}{3}\right)=\frac{\sqrt3}{2}$$
Final Answer
$$\sin\!\left(\cos^{-1}\frac{1}{2}\right)=\frac{\sqrt3}{2}$$

Problem 11

Evaluate Composite Inverse Trig
Find the exact value
$$\tan\!\left(\sin^{-1}\!\left(-\frac{1}{2}\right)\right)$$
Step 1 — Inner function
$\sin(-\pi/6)=-\tfrac{1}{2}$ and $-\pi/6 \in [-\pi/2,\pi/2]$, so $\sin^{-1}(-\tfrac{1}{2})=-\dfrac{\pi}{6}$.
Step 2 — Outer function
$$\tan\!\left(-\frac{\pi}{6}\right)=-\tan\!\frac{\pi}{6}=-\frac{\sqrt3}{3}$$
Final Answer
$$\tan\!\left(\sin^{-1}\!\left(-\tfrac{1}{2}\right)\right)=-\frac{\sqrt3}{3}$$

Problem 12

Evaluate Composite Inverse Trig
Find the exact value
$$\tan\!\left(\cos^{-1}\!\left(-\frac{\sqrt3}{2}\right)\right)$$
Step 1 — Inner function
$\cos(5\pi/6)=-\dfrac{\sqrt3}{2}$ and $5\pi/6 \in [0,\pi]$, so $\cos^{-1}\!\left(-\dfrac{\sqrt3}{2}\right)=\dfrac{5\pi}{6}$.
Step 2 — Outer function
$$\tan\!\left(\frac{5\pi}{6}\right)=\frac{\sin(5\pi/6)}{\cos(5\pi/6)}=\frac{1/2}{-\sqrt3/2}=-\frac{1}{\sqrt3}=-\frac{\sqrt3}{3}$$
Final Answer
$$\tan\!\left(\cos^{-1}\!\left(-\tfrac{\sqrt3}{2}\right)\right)=-\frac{\sqrt3}{3}$$

Problem 13

Evaluate Composite Inverse Trig
Find the exact value
$$\cot\!\left[\sin^{-1}\!\left(-\frac{1}{2}\right)\right]$$
Step 1 — Inner function
$\sin^{-1}(-\tfrac{1}{2})=-\dfrac{\pi}{6}$ (from Problem 11).
Step 2 — Outer function
$$\cot\!\left(-\frac{\pi}{6}\right)=\frac{\cos(-\pi/6)}{\sin(-\pi/6)}=\frac{\sqrt3/2}{-1/2}=-\sqrt3$$
Final Answer
$$\cot\!\left[\sin^{-1}\!\left(-\tfrac{1}{2}\right)\right]=-\sqrt3$$

Problem 14

Evaluate Composite Inverse Trig
Find the exact value
$$\sec\!\left[\cos^{-1}\frac{1}{2}\right]$$
Step 1 — Inner function
$\cos^{-1}\!\tfrac{1}{2}=\dfrac{\pi}{3}$ (from Problem 10).
Step 2 — Outer function
$$\sec\!\left(\frac{\pi}{3}\right)=\frac{1}{\cos(\pi/3)}=\frac{1}{1/2}=2$$
Final Answer
$$\sec\!\left[\cos^{-1}\frac{1}{2}\right]=2$$

Problem 15

Evaluate Composite Inverse Trig
Find the exact value
$$\sec\!\left(\tan^{-1}\sqrt3\right)$$
Step 1 — Inner function
$\tan(\pi/3)=\sqrt3$ and $\pi/3 \in (-\pi/2,\pi/2)$, so $\tan^{-1}\!\sqrt3=\dfrac{\pi}{3}$.
Step 2 — Outer function
$$\sec\!\left(\frac{\pi}{3}\right)=\frac{1}{\cos(\pi/3)}=\frac{1}{1/2}=2$$
Final Answer
$$\sec\!\left(\tan^{-1}\sqrt3\right)=2$$

Problem 16

Evaluate Composite Inverse Trig
Find the exact value
$$\csc\!\left(\tan^{-1}1\right)$$
Step 1 — Inner function
$\tan(\pi/4)=1$ and $\pi/4 \in (-\pi/2,\pi/2)$, so $\tan^{-1}1=\dfrac{\pi}{4}$.
Step 2 — Outer function
$$\csc\!\left(\frac{\pi}{4}\right)=\frac{1}{\sin(\pi/4)}=\frac{1}{\sqrt2/2}=\sqrt2$$
Final Answer
$$\csc\!\left(\tan^{-1}1\right)=\sqrt2$$

Problem 17

Evaluate Composite Inverse Trig
Find the exact value
$$\cos\!\left[\sin^{-1}\!\left(-\frac{\sqrt3}{2}\right)\right]$$
Step 1 — Inner function
$\sin(-\pi/3)=-\dfrac{\sqrt3}{2}$ and $-\pi/3 \in [-\pi/2,\pi/2]$, so $\sin^{-1}\!\left(-\dfrac{\sqrt3}{2}\right)=-\dfrac{\pi}{3}$.
Step 2 — Outer function
$$\cos\!\left(-\frac{\pi}{3}\right)=\cos\!\frac{\pi}{3}=\frac{1}{2}$$
Final Answer
$$\cos\!\left[\sin^{-1}\!\left(-\tfrac{\sqrt3}{2}\right)\right]=\frac{1}{2}$$

Problem 18

Evaluate Composite Inverse Trig
Find the exact value
$$\sin\!\left[\tan^{-1}(-1)\right]$$
Step 1 — Inner function
$\tan(-\pi/4)=-1$ and $-\pi/4 \in (-\pi/2,\pi/2)$, so $\tan^{-1}(-1)=-\dfrac{\pi}{4}$.
Step 2 — Outer function
$$\sin\!\left(-\frac{\pi}{4}\right)=-\sin\!\frac{\pi}{4}=-\frac{\sqrt2}{2}$$
Final Answer
$$\sin\!\left[\tan^{-1}(-1)\right]=-\frac{\sqrt2}{2}$$

Problem 19

Evaluate Composite Inverse Trig
Find the exact value
$$\csc\!\left[\cos^{-1}\!\left(-\frac{\sqrt3}{2}\right)\right]$$
Step 1 — Inner function
$\cos^{-1}\!\left(-\dfrac{\sqrt3}{2}\right)=\dfrac{5\pi}{6}$ (from Problem 12).
Step 2 — Outer function
$$\csc\!\left(\frac{5\pi}{6}\right)=\frac{1}{\sin(5\pi/6)}=\frac{1}{1/2}=2$$
Final Answer
$$\csc\!\left[\cos^{-1}\!\left(-\tfrac{\sqrt3}{2}\right)\right]=2$$

Problem 20

Evaluate Composite Inverse Trig
Find the exact value
$$\sec\!\left[\sin^{-1}\!\left(-\frac{1}{2}\right)\right]$$
Step 1 — Inner function
$\sin^{-1}(-\tfrac{1}{2})=-\dfrac{\pi}{6}$ (from Problem 11).
Step 2 — Outer function
$$\sec\!\left(-\frac{\pi}{6}\right)=\frac{1}{\cos(-\pi/6)}=\frac{1}{\cos(\pi/6)}=\frac{1}{\sqrt3/2}=\frac{2}{\sqrt3}=\frac{2\sqrt3}{3}$$
Final Answer
$$\sec\!\left[\sin^{-1}\!\left(-\tfrac{1}{2}\right)\right]=\frac{2\sqrt3}{3}$$
