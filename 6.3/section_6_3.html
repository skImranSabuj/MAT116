★ Key Notes for Section 6.3
Pythagorean Identity — Core Tool
$$\sin^2\theta + \cos^2\theta = 1$$ Use it to find the missing primary function, then derive the rest: $\tan\theta=\dfrac{\sin\theta}{\cos\theta}$, $\csc\theta=\dfrac{1}{\sin\theta}$, $\sec\theta=\dfrac{1}{\cos\theta}$, $\cot\theta=\dfrac{1}{\tan\theta}$
Signs in Each Quadrant (ASTC rule)
QI: All positive QII: sin, csc positive QIII: tan, cot positive QIV: cos, sec positive Memory: "All Students Take Calculus" → QI→QII→QIII→QIV
Strategy
1. Identify the quadrant from the given conditions. 2. Use the Pythagorean identity to find the missing trig value with correct sign. 3. Compute the remaining four functions using definitions.

Problem 43

Find Remaining Trig Functions
Given
$\sin\theta = \dfrac{12}{13}$,   $\theta$ in Quadrant II
Step 1 — Find $\cos\theta$ via Pythagorean identity
$$\cos^2\theta = 1-\sin^2\theta = 1-\left(\frac{12}{13}\right)^2 = 1-\frac{144}{169}=\frac{25}{169}$$ In QII, $\cos\theta \lt 0$: $$\cos\theta = -\frac{5}{13}$$
Step 2 — Compute all six functions
$$\tan\theta = \frac{\sin\theta}{\cos\theta}=\frac{12/13}{-5/13}=-\frac{12}{5}$$ $$\csc\theta = \frac{1}{\sin\theta}=\frac{13}{12}$$ $$\sec\theta = \frac{1}{\cos\theta}=-\frac{13}{5}$$ $$\cot\theta = \frac{1}{\tan\theta}=-\frac{5}{12}$$
Final Answer
$$\cos\theta=-\tfrac{5}{13},\quad\tan\theta=-\tfrac{12}{5},\quad\csc\theta=\tfrac{13}{12},\quad\sec\theta=-\tfrac{13}{5},\quad\cot\theta=-\tfrac{5}{12}$$

Problem 44

Find Remaining Trig Functions
Given
$\cos\theta = \dfrac{3}{5}$,   $\theta$ in Quadrant IV
Step 1 — Find $\sin\theta$
$$\sin^2\theta = 1-\left(\frac{3}{5}\right)^2=1-\frac{9}{25}=\frac{16}{25}$$ In QIV, $\sin\theta \lt 0$: $\quad\sin\theta = -\dfrac{4}{5}$
Step 2 — Remaining functions
$$\tan\theta = \frac{-4/5}{3/5}=-\frac{4}{3},\quad\csc\theta=-\frac{5}{4},\quad\sec\theta=\frac{5}{3},\quad\cot\theta=-\frac{3}{4}$$
Final Answer
$$\sin\theta=-\tfrac{4}{5},\quad\tan\theta=-\tfrac{4}{3},\quad\csc\theta=-\tfrac{5}{4},\quad\sec\theta=\tfrac{5}{3},\quad\cot\theta=-\tfrac{3}{4}$$

Problem 45

Find Remaining Trig Functions
Given
$\sin\theta = -\dfrac{5}{13}$,   $\theta$ in Quadrant III
Step 1 — Find $\cos\theta$
$$\cos^2\theta=1-\frac{25}{169}=\frac{144}{169}$$ In QIII, $\cos\theta \lt 0$: $\quad\cos\theta=-\dfrac{12}{13}$
Step 2 — Remaining functions
$$\tan\theta=\frac{-5/13}{-12/13}=\frac{5}{12},\quad\csc\theta=-\frac{13}{5},\quad\sec\theta=-\frac{13}{12},\quad\cot\theta=\frac{12}{5}$$
Final Answer
$$\cos\theta=-\tfrac{12}{13},\quad\tan\theta=\tfrac{5}{12},\quad\csc\theta=-\tfrac{13}{5},\quad\sec\theta=-\tfrac{13}{12},\quad\cot\theta=\tfrac{12}{5}$$

Problem 49

Find Remaining Trig Functions
Given
$\sin\theta = -\dfrac{2}{3}$,   $\pi \lt \theta \lt \dfrac{3\pi}{2}$ (Quadrant III)
Step 1 — Find $\cos\theta$
$$\cos^2\theta=1-\frac{4}{9}=\frac{5}{9}$$ In QIII, $\cos\theta \lt 0$: $\quad\cos\theta=-\dfrac{\sqrt5}{3}$
Step 2 — Remaining functions
$$\tan\theta=\frac{-2/3}{-\sqrt5/3}=\frac{2}{\sqrt5}=\frac{2\sqrt5}{5}$$ $$\csc\theta=-\frac{3}{2},\quad\sec\theta=-\frac{3}{\sqrt5}=-\frac{3\sqrt5}{5},\quad\cot\theta=\frac{\sqrt5}{2}$$
Final Answer
$$\cos\theta=-\frac{\sqrt5}{3},\quad\tan\theta=\frac{2\sqrt5}{5},\quad\csc\theta=-\frac{3}{2},\quad\sec\theta=-\frac{3\sqrt5}{5},\quad\cot\theta=\frac{\sqrt5}{2}$$

Problem 50

Find Remaining Trig Functions
Given
$\cos\theta = -\dfrac{1}{3}$,   $\dfrac{\pi}{2} \lt \theta \lt \pi$ (Quadrant II)
Step 1 — Find $\sin\theta$
$$\sin^2\theta=1-\frac{1}{9}=\frac{8}{9}$$ In QII, $\sin\theta \gt 0$: $\quad\sin\theta=\dfrac{2\sqrt2}{3}$
Step 2 — Remaining functions
$$\tan\theta=\frac{2\sqrt2/3}{-1/3}=-2\sqrt2$$ $$\csc\theta=\frac{3}{2\sqrt2}=\frac{3\sqrt2}{4},\quad\sec\theta=-3,\quad\cot\theta=\frac{-1/3}{2\sqrt2/3}=-\frac{1}{2\sqrt2}=-\frac{\sqrt2}{4}$$
Final Answer
$$\sin\theta=\frac{2\sqrt2}{3},\quad\tan\theta=-2\sqrt2,\quad\csc\theta=\frac{3\sqrt2}{4},\quad\sec\theta=-3,\quad\cot\theta=-\frac{\sqrt2}{4}$$

Problem 51

Find Remaining Trig Functions
Given
$\cos\theta = -\dfrac{1}{4}$,   $\tan\theta \gt 0$
Step 1 — Identify the quadrant
$\cos\theta \lt 0$ and $\tan\theta \gt 0 \;\Rightarrow\;$ both $\sin\theta$ and $\cos\theta$ must be negative $\Rightarrow$ Quadrant III.
Step 2 — Find $\sin\theta$
$$\sin^2\theta=1-\frac{1}{16}=\frac{15}{16}$$ In QIII: $\quad\sin\theta=-\dfrac{\sqrt{15}}{4}$
Step 3 — Remaining functions
$$\tan\theta=\frac{-\sqrt{15}/4}{-1/4}=\sqrt{15}$$ $$\csc\theta=-\frac{4}{\sqrt{15}}=-\frac{4\sqrt{15}}{15},\quad\sec\theta=-4,\quad\cot\theta=\frac{1}{\sqrt{15}}=\frac{\sqrt{15}}{15}$$
Final Answer
$$\sin\theta=-\frac{\sqrt{15}}{4},\quad\tan\theta=\sqrt{15},\quad\csc\theta=-\frac{4\sqrt{15}}{15},\quad\sec\theta=-4,\quad\cot\theta=\frac{\sqrt{15}}{15}$$

Problem 52

Find Remaining Trig Functions
Given
$\sin\theta = \dfrac{2}{3}$,   $\tan\theta \lt 0$
Step 1 — Identify the quadrant
$\sin\theta \gt 0$ and $\tan\theta \lt 0 \;\Rightarrow\; \cos\theta \lt 0 \;\Rightarrow$ Quadrant II.
Step 2 — Find $\cos\theta$
$$\cos^2\theta=1-\frac{4}{9}=\frac{5}{9}$$ In QII: $\quad\cos\theta=-\dfrac{\sqrt5}{3}$
Step 3 — Remaining functions
$$\tan\theta=\frac{2/3}{-\sqrt5/3}=-\frac{2}{\sqrt5}=-\frac{2\sqrt5}{5}$$ $$\csc\theta=\frac{3}{2},\quad\sec\theta=-\frac{3}{\sqrt5}=-\frac{3\sqrt5}{5},\quad\cot\theta=-\frac{\sqrt5}{2}$$
Final Answer
$$\cos\theta=-\frac{\sqrt5}{3},\quad\tan\theta=-\frac{2\sqrt5}{5},\quad\csc\theta=\frac{3}{2},\quad\sec\theta=-\frac{3\sqrt5}{5},\quad\cot\theta=-\frac{\sqrt5}{2}$$

Problem 53

Find Remaining Trig Functions
Given
$\csc\theta = 3$,   $\cot\theta \lt 0$
Step 1 — Extract $\sin\theta$ and identify quadrant
$\csc\theta=3 \Rightarrow \sin\theta=\dfrac{1}{3} \gt 0$. $\cot\theta \lt 0 \Rightarrow \cos\theta/\sin\theta \lt 0$. Since $\sin\theta \gt 0$, we need $\cos\theta \lt 0 \Rightarrow$ Quadrant II.
Step 2 — Find $\cos\theta$
$$\cos^2\theta=1-\frac{1}{9}=\frac{8}{9}$$ In QII: $\quad\cos\theta=-\dfrac{2\sqrt2}{3}$
Step 3 — Remaining functions
$$\tan\theta=\frac{1/3}{-2\sqrt2/3}=-\frac{1}{2\sqrt2}=-\frac{\sqrt2}{4}$$ $$\sec\theta=-\frac{3}{2\sqrt2}=-\frac{3\sqrt2}{4},\quad\cot\theta=-2\sqrt2$$
Final Answer
$$\sin\theta=\tfrac{1}{3},\;\cos\theta=-\tfrac{2\sqrt2}{3},\;\tan\theta=-\tfrac{\sqrt2}{4},\;\sec\theta=-\tfrac{3\sqrt2}{4},\;\cot\theta=-2\sqrt2$$

Problem 54

Find Remaining Trig Functions
Given
$\sec\theta = 2$,   $\sin\theta \lt 0$
Step 1 — Extract $\cos\theta$ and identify quadrant
$\sec\theta=2 \Rightarrow \cos\theta=\dfrac{1}{2} \gt 0$. $\cos\theta \gt 0$ and $\sin\theta \lt 0 \Rightarrow$ Quadrant IV.
Step 2 — Find $\sin\theta$
$$\sin^2\theta=1-\frac{1}{4}=\frac{3}{4}$$ In QIV: $\quad\sin\theta=-\dfrac{\sqrt3}{2}$
Step 3 — Remaining functions
$$\tan\theta=\frac{-\sqrt3/2}{1/2}=-\sqrt3$$ $$\csc\theta=-\frac{2}{\sqrt3}=-\frac{2\sqrt3}{3},\quad\cot\theta=-\frac{1}{\sqrt3}=-\frac{\sqrt3}{3}$$
Final Answer
$$\sin\theta=-\tfrac{\sqrt3}{2},\;\cos\theta=\tfrac{1}{2},\;\tan\theta=-\sqrt3,\;\csc\theta=-\tfrac{2\sqrt3}{3},\;\cot\theta=-\tfrac{\sqrt3}{3}$$

Problem 55

Find Remaining Trig Functions
Given
$\cot\theta = \dfrac{4}{3}$,   $\cos\theta \lt 0$
Step 1 — Identify the quadrant
$\cot\theta \gt 0 \Rightarrow \tan\theta \gt 0 \Rightarrow \sin$ and $\cos$ have the same sign. $\cos\theta \lt 0 \Rightarrow \sin\theta \lt 0$ as well $\Rightarrow$ Quadrant III.
Step 2 — Use reference triangle: opposite $=3$, adjacent $=4$, hypotenuse $=5$
In QIII both components are negative: $$\sin\theta=-\frac{3}{5},\quad\cos\theta=-\frac{4}{5}$$ Check: $\cot\theta=\cos\theta/\sin\theta=(-4/5)/(-3/5)=4/3$ ✓
Step 3 — Remaining functions
$$\tan\theta=\frac{3}{4},\quad\csc\theta=-\frac{5}{3},\quad\sec\theta=-\frac{5}{4}$$
Final Answer
$$\sin\theta=-\tfrac{3}{5},\;\cos\theta=-\tfrac{4}{5},\;\tan\theta=\tfrac{3}{4},\;\csc\theta=-\tfrac{5}{3},\;\sec\theta=-\tfrac{5}{4}$$
