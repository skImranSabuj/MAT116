★ Key Notes for Section 6.2
Periodicity — the Core Strategy
Reduce any large angle by subtracting full periods until the angle is in $[0,2\pi)$: $\sin\theta$, $\cos\theta$: period $= 2\pi$ $\tan\theta$, $\cot\theta$: period $= \pi$ $\sec\theta$, $\csc\theta$: period $= 2\pi$ Then use standard unit-circle values.
Quick Unit-Circle Reference
$\sin 0=0,\;\sin\tfrac{\pi}{2}=1,\;\sin\pi=0,\;\sin\tfrac{3\pi}{2}=-1$ $\cos 0=1,\;\cos\tfrac{\pi}{2}=0,\;\cos\pi=-1,\;\cos\tfrac{3\pi}{2}=0$ Negative angles: $\sin(-\alpha)=-\sin\alpha$, $\cos(-\alpha)=\cos\alpha$ (even/odd)

Problem 21

Exact Value — Unit Circle
Given
Find the exact value of $\;\sin\!\dfrac{11\pi}{2}$
Step 1 — Reduce using period $2\pi$
Divide $\frac{11\pi}{2}$ by $2\pi$: $\quad \frac{11\pi/2}{2\pi}=\frac{11}{4}=2\tfrac{3}{4}$ So $\dfrac{11\pi}{2} = 2(2\pi)+\dfrac{3\pi}{2}$. Therefore $\sin\!\dfrac{11\pi}{2}=\sin\!\dfrac{3\pi}{2}$.
Step 2 — Read unit-circle value
At $\theta=\dfrac{3\pi}{2}$ (bottom of unit circle): $\sin\!\dfrac{3\pi}{2}=-1$.
Final Answer
$$\sin\frac{11\pi}{2} = -1$$

Problem 22

Exact Value — Unit Circle
Given
Find the exact value of $\;\cos(7\pi)$
Step 1 — Reduce: $7\pi = 6\pi + \pi = 3(2\pi)+\pi$
$$\cos(7\pi)=\cos(\pi)$$
Step 2 — Unit-circle value
$$\cos(\pi)=-1$$
Final Answer
$$\cos(7\pi) = -1$$

Problem 23

Exact Value — Unit Circle
Given
Find the exact value of $\;\cot\!\dfrac{7\pi}{2}$
Step 1 — Reduce using period $2\pi$
$\dfrac{7\pi}{2} = 2\pi + \dfrac{3\pi}{2}$, so $\cot\!\dfrac{7\pi}{2}=\cot\!\dfrac{3\pi}{2}$.
Step 2 — Evaluate using definition
$$\cot\!\frac{3\pi}{2}=\frac{\cos(3\pi/2)}{\sin(3\pi/2)}=\frac{0}{-1}=0$$
Final Answer
$$\cot\frac{7\pi}{2} = 0$$

Problem 24

Exact Value — Unit Circle
Given
Find the exact value of $\;\tan(6\pi)$
Step 1 — $6\pi = 3(2\pi)$, so reduce to $0$
$$\tan(6\pi)=\tan(0)$$
Step 2 — Unit-circle value
$$\tan(0)=\frac{\sin 0}{\cos 0}=\frac{0}{1}=0$$
Final Answer
$$\tan(6\pi) = 0$$

Problem 25

Exact Value — Unit Circle
Given
Find the exact value of $\;\sec(8\pi)$
Step 1 — Reduce: $8\pi = 4(2\pi) \to$ angle $0$
$$\sec(8\pi)=\sec(0)$$
Step 2 — Evaluate
$$\sec(0)=\frac{1}{\cos 0}=\frac{1}{1}=1$$
Final Answer
$$\sec(8\pi) = 1$$

Problem 26

Exact Value — Unit Circle
Given
Find the exact value of $\;\csc\!\dfrac{11\pi}{2}$
Step 1 — Reduce (same as Problem 21)
$\dfrac{11\pi}{2}=2(2\pi)+\dfrac{3\pi}{2}$, so $\csc\!\dfrac{11\pi}{2}=\csc\!\dfrac{3\pi}{2}$.
Step 2 — Evaluate
$$\csc\!\frac{3\pi}{2}=\frac{1}{\sin(3\pi/2)}=\frac{1}{-1}=-1$$
Final Answer
$$\csc\frac{11\pi}{2} = -1$$

Problem 27

Exact Value — Unit Circle
Given
Find the exact value of $\;\sin(-3\pi)$
Step 1 — Use odd symmetry: $\sin(-\alpha)=-\sin(\alpha)$
$$\sin(-3\pi)=-\sin(3\pi)$$
Step 2 — Reduce $3\pi$: $3\pi=2\pi+\pi \Rightarrow \sin(3\pi)=\sin(\pi)=0$
$$-\sin(3\pi)=-0=0$$
Final Answer
$$\sin(-3\pi) = 0$$

Problem 28

Exact Value — Unit Circle
Given
Find the exact value of $\;\cos\!\left(-\dfrac{3\pi}{2}\right)$
Step 1 — Use even symmetry: $\cos(-\alpha)=\cos(\alpha)$
$$\cos\!\left(-\frac{3\pi}{2}\right)=\cos\!\frac{3\pi}{2}$$
Step 2 — Unit-circle value
At $\theta=\dfrac{3\pi}{2}$: the point on the unit circle is $(0,-1)$, so $\cos\!\dfrac{3\pi}{2}=0$.
Final Answer
$$\cos\!\left(-\frac{3\pi}{2}\right) = 0$$
