★ Key Notes for this Section
Logarithmic Function Definition
$y = \log_a x \iff a^y = x$   ($a > 0$, $a \neq 1$, $x > 0$) Common log: $\log x = \log_{10} x$ Natural log: $\ln x = \log_e x$
Properties of y = log_a x
• Domain: $(0,\infty)$; Range: $(-\infty,\infty)$ • Vertical asymptote: $x = 0$; passes through $(1,0)$ and $(a,1)$ • If $a > 1$: increasing; if $0 < a < 1$: decreasing • $\log_a(a^x) = x$ and $a^{\log_a x} = x$
Key Identities
$\log_a 1 = 0$   $\log_a a = 1$   $\ln e = 1$ $\log_a a^r = r$   $a^{\log_a x} = x$ Log and exponential are inverse functions: $f(x) = a^x$ and $f^{-1}(x) = \log_a x$

State characteristics
Domain: $(2,\infty)$ Range: $(-\infty,\,\infty)$ Vertical asymptote: $x = 2$

Identify the transformation

State characteristics
Domain: $(2,\infty)$ Range: $(-\infty,\,\infty)$ Vertical asymptote: $x = 2$
Final Answer

Exam Answer
Solve x² - 1 > 0

Final Answer
Domain: $(-\infty,-1)\cup(1,\infty)$
