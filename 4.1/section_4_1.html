Problem 37

Graphing with Transformations
Given
f(x) = (x + 2)^4 - 3
Step 1 - Start with the Parent Function (Ignore the numbers)
Just look at the POWER for a moment - ignore everything else. Parent function: y = x^4 What does y = x^4 look like? Shape : U-shaped (like a bowl) Opens : upward Key Pt : Vertex at (0, 0) Symmetry : symmetric about the y-axis
Step 2 - Understand the Changes (Transformations)
Now compare y = x^4 with the given function: $$f(x) = 1\left(x - (-2)\right)^{4} - 3$$ There are 2 change(s) from the parent: Change 1: SHIFT LEFT by 2 units You see (x + 2) inside the bracket. Trick: PLUS inside = move LEFT. (Feels backwards - but it is correct!) Change 2: SHIFT DOWN by 3 units The -3 is OUTSIDE the bracket, so the whole graph moves down.
Step 3 - Find the New Vertex
The parent's key point starts at (0, 0). Apply the shifts one at a time: Horizontal shift: move left 2 unit(s) -> x becomes -2 Vertical shift : move down 3 unit(s) -> y becomes -3 New Vertex = (-2, -3) This is the LOWEST point on the graph.
Step 4 - Shape of the Graph
The power is 4, which is EVEN. Even power (+ a = 1): -> Graph opens UPWARD (U-shape) -> Symmetric about the vertical line x = -2
Step 5 - y-intercept (where does the graph cross the y-axis?)
Plug in x = 0: $$f(0) = 1(0 - (-2))^{4} - 3$$ $$f(0) = 1 \times 16 - 3$$ $$f(0) = 16 - 3 = 13$$ y-intercept = (0, 13)
Step 6 - End Behavior (what happens at the far left and right?)
Look at the leading term to decide: As $x \to -\infty,\ f(x) \to +\infty$ and as $x \to +\infty,\ f(x) \to +\infty$ In plain words: The graph rises on BOTH sides (like a U-shape).
Step 7 - Final Summary (copy this into your exam answer!)
Parent Function : y = x^4 Transformation : SHIFT LEFT by 2 units, SHIFT DOWN by 3 units Vertex : (-2, -3) y-intercept : (0, 13) Shape : U-shape, opens UP End Behavior : The graph rises on BOTH sides (like a U-shape).
Graph of f(x)
Final Answer Parent: $y = x^{4}$ | Vertex: (-2, -3) | y-intercept: (0, 13) | The graph rises on BOTH sides (like a U-shape).

Problem 43

Form a Polynomial
Given
Zeros: -1, 1, 3; degree 3
Step 1 - What Does a 'Zero' Mean?
A zero (or root) is an x-value that makes f(x) = 0. Visually: the graph CROSSES or TOUCHES the x-axis at each zero. The Factor Theorem says: If x = r is a zero, then (x - r) is a FACTOR of f(x). Quick examples: Zero at x = 3 -> factor is (x - 3) Zero at x = -2 -> factor is (x - (-2)) = (x + 2) Zero at x = 0 -> factor is (x - 0) = x
Step 2 - Convert Each Zero to a Factor
Apply the Factor Theorem to every zero: Zero x = -1 -> factor: (x + 1) Zero x = 1 -> factor: (x - 1) Zero x = 3 -> factor: (x - 3)
Step 3 - Write the Factored Form
Multiply all factors together. Use leading coefficient a = 1: $$f(x) = (x + 1) \cdot (x - 1) \cdot (x - 3)$$ Note: We use a = 1 because the problem does not specify a value. Any non-zero multiple of this polynomial has the same zeros.
Step 4 - Expand to Standard Form
Multiply out all the brackets (FOIL or distributive property): $$f(x) = x^3 - 3x^2 - x + 3$$
Step 5 - Verify the Degree
Add up all the multiplicities - this must equal the stated degree: x = -1, multiplicity 1 x = 1, multiplicity 1 x = 3, multiplicity 1 Total = 3 Degree = 3 (matches!)
Graph of f(x)
Final Answer $f(x) = (x + 1) \cdot (x - 1) \cdot (x - 3) = x^3 - 3x^2 - x + 3$
