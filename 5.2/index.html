Problem 39

Inverse_verify
Given
Verify inverses: f(x)=x^3-8, g(x)=cuberoot(x+8).
Check f(g(x))
f(g(x)) = (cuberoot(x+8))^3 - 8 = x
Check g(f(x))
g(f(x)) = cuberoot((x^3-8)+8) = cuberoot(x^3) = x
Exam Answer
f and g are inverses on all real numbers.

Problem 40

Inverse_verify
Given
Verify inverses: f(x)=sqrt(x-2), x>=2; g(x)=x^2+2.
Check f(g(x))
f(g(x)) = sqrt((x^2+2)-2) = sqrt(x^2) = x for x>=0
Check g(f(x))
g(f(x)) = (sqrt(x-2))^2 + 2 = x for x>=2
Exam Answer
They are inverses with f-domain [2,inf) and g-domain [0,inf).

Problem 66

Inverse_find
Given
Find f^{-1} for f(x)=3x/(x+2), and state domain/range of f and f^{-1}.
Solve y=3x/(x+2) for x
y(x+2)=3x -> yx+2y=3x -> x(y-3)=-2y -> x=2y/(3-y).
Write inverse and sets
f^{-1}(x)=2x/(3-x). Dom(f): x!= -2, Range(f): y!=3. Dom(f^{-1}): x!=3, Range(f^{-1}): y!=-2.
Exam Answer
f^{-1}(x)=2x/(3-x); Dom(f)=R\{-2}, Range(f)=R\{3}.

Problem 73

Inverse_find
Given
Find f^{-1} for f(x)=(x^2+3)/(3x^2), x>0, and state domain/range of f and f^{-1}.
Rewrite and solve
y = 1/3 + 1/x^2 -> y-1/3 = 1/x^2 -> x^2 = 3/(3y-1). Since x>0, x = sqrt(3/(3y-1)).
Write inverse and sets
f^{-1}(x)=sqrt(3/(3x-1)). Dom(f)=(0,inf), Range(f)=(1/3,inf). Dom(f^{-1})=(1/3,inf), Range(f^{-1})=(0,inf).
Exam Answer
f^{-1}(x)=sqrt(3/(3x-1)), x>1/3.

Problem 74

Inverse_find
Given
Find f^{-1} for f(x)=(x^2-4)/(2x^2), x>0, and state domain/range of f and f^{-1}.
Rewrite and solve
y = 1/2 - 2/x^2 -> 1/2-y = 2/x^2 -> x^2 = 4/(1-2y). Since x>0, x = 2/sqrt(1-2y).
Write inverse and sets
f^{-1}(x)=2/sqrt(1-2x). Dom(f)=(0,inf), Range(f)=(-inf,1/2). Dom(f^{-1})=(-inf,1/2), Range(f^{-1})=(0,inf).
Exam Answer
f^{-1}(x)=2/sqrt(1-2x), x<1/2.
