★ Key Notes for this Section
One-to-One Function
A function $f$ is one-to-one if $f(a) = f(b)$ implies $a = b$. Graphically: passes the Horizontal Line Test. Only one-to-one functions have an inverse $f^{-1}$.
Inverse Function Property
$(f^{-1} \circ f)(x) = x$ for all $x$ in Dom($f$) $(f \circ f^{-1})(x) = x$ for all $x$ in Dom($f^{-1}$) Dom($f^{-1}$) = Range($f$) and Range($f^{-1}$) = Dom($f$).
Procedure — Finding f⁻¹
1. Write $y = f(x)$. 2. Swap $x$ and $y$: solve the equation $x = f(y)$ for $y$. 3. Write $f^{-1}(x) = $ (result). 4. State domain restrictions.

Problem 39

Verify Inverse Pair
Given
Verify that $f(x) = x^3 - 8$ and $g(x) = \sqrt[3]{x+8}$ are inverses.
Check f(g(x)) = x
$f(g(x)) = \left(\sqrt[3]{x+8}\right)^3 - 8 = (x+8) - 8 = x$ ✓
Check g(f(x)) = x
$g(f(x)) = \sqrt[3]{(x^3-8)+8} = \sqrt[3]{x^3} = x$ ✓
Exam Answer
Check f(g(x)) = x
$f(g(x)) = \left(\sqrt[3]{x+8}\right)^3 - 8 = (x+8) - 8 = x$ ✓
Check g(f(x)) = x
$g(f(x)) = \sqrt[3]{(x^3-8)+8} = \sqrt[3]{x^3} = x$ ✓
Final Answer
Since both compositions equal $x$, $f$ and $g$ are inverse functions on $(-\infty,\infty)$.

Final Answer
They are inverse functions with Dom($f$) $= [2,\infty)$, Dom($g$) $= [0,\infty)$.

State domain restrictions
Dom($f$): $x \neq -\dfrac{3}{2}$; Dom($g$): $x \neq \dfrac{1}{2}$
Exam Answer
Check f(g(x)) = x

Problem 73

Find Inverse Function
Given
Find $f^{-1}$ for $f(x) = \dfrac{x^2+3}{3x^2}$, $x > 0$. State domain and range.
Rewrite and solve for x
$y = \dfrac{1}{3} + \dfrac{1}{x^2} \Rightarrow y - \dfrac{1}{3} = \dfrac{1}{x^2} \Rightarrow x^2 = \dfrac{1}{y-\frac{1}{3}} = \dfrac{3}{3y-1}$ Since $x > 0$: $x = \sqrt{\dfrac{3}{3y-1}}$
Write inverse and state sets
$f^{-1}(x) = \sqrt{\dfrac{3}{3x-1}}$ Dom($f$) $= (0,\infty)$; Range($f$) $= \left(\dfrac{1}{3},\infty\right)$ Dom($f^{-1}$) $= \left(\dfrac{1}{3},\infty\right)$; Range($f^{-1}$) $= (0,\infty)$
Exam Answer

$y = \dfrac{1}{3} + \dfrac{1}{x^2} \Rightarrow y - \dfrac{1}{3} = \dfrac{1}{x^2} \Rightarrow x^2 = \dfrac{1}{y-\frac{1}{3}} = \dfrac{3}{3y-1}$ Since $x > 0$: $x = \sqrt{\dfrac{3}{3y-1}}$
Write inverse and state sets
$f^{-1}(x) = \sqrt{\dfrac{3}{3x-1}}$ Dom($f$) $= (0,\infty)$; Range($f$) $= \left(\dfrac{1}{3},\infty\right)$ Dom($f^{-1}$) $= \left(\dfrac{1}{3},\infty\right)$; Range($f^{-1}$) $= (0,\infty)$
