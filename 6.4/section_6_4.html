★ Key Notes for Section 6.4
General Form
$$y = A\sin(\omega x + \phi) + B \qquad \text{or} \qquad y = A\cos(\omega x + \phi) + B$$ Amplitude $= |A|$  (negative $A$ reflects the graph over the $x$-axis) Period $= \dfrac{2\pi}{\omega}$ Phase Shift $= -\dfrac{\phi}{\omega}$ Vertical Shift $= B$ (shifts midline to $y=B$) Range $= [B-|A|,\; B+|A|]$
Five Key Points per Cycle
Divide one full period into 4 equal sub-intervals. Evaluate at each quarter-period mark. For $y=\sin$: start at 0, peak at T/4, return to 0 at T/2, trough at 3T/4, return at T. For $y=\cos$: start at max, cross midline at T/4, trough at T/2, midline at 3T/4, max at T.

Problem 41

Graph using Transformations
Given
$$y = 2\cos\!\left(\frac{x}{4}\right)$$ Graph using transformations. State domain and range.
Step 1 — Identify parameters
$A=2$, $\;\omega=\dfrac{1}{4}$, $\;\phi=0$, $\;B=0$. Amplitude $= 2$,   Period $= \dfrac{2\pi}{1/4} = 8\pi$.
Step 2 — Five key points (one full cycle, starting at $x=0$)
$x$ $0$ $2\pi$ $4\pi$ $6\pi$ $8\pi$
$y$ $2$ $0$ $-2$ $0$ $2$
Step 3 — Transformation description
Start from $y=\cos x$. ① Horizontal stretch by factor 4 (period: $2\pi \to 8\pi$). ② Vertical stretch by factor 2 (amplitude: $1 \to 2$).
Graph: y = 2 cos(x/4) — two full cycles
Final Answer
Amplitude $= 2$,   Period $= 8\pi \approx 25.13$ Domain: $(-\infty, \infty)$,   Range: $[-2, 2]$

Problem 42

Graph using Transformations
Given
$$y = 2\sin\!\left(\frac{x}{2}\right)$$
Step 1 — Parameters
$A=2$, $\omega=\frac{1}{2}$. Amplitude $= 2$,   Period $= \dfrac{2\pi}{1/2} = 4\pi$.
Step 2 — Key points (one cycle)
$x$ $0$ $\pi$ $2\pi$ $3\pi$ $4\pi$
$y$ $0$ $2$ $0$ $-2$ $0$
Step 3 — Transformation
Start from $y=\sin x$: horizontal stretch by 2, then vertical stretch by 2.
Graph: y = 2 sin(x/2) — two full cycles
Final Answer
Amplitude $= 2$,   Period $= 4\pi \approx 12.57$ Domain: $(-\infty,\infty)$,   Range: $[-2,2]$

Problem 43

Graph using Transformations
Given
$$y = -\frac{1}{2}\cos(2x)$$
Step 1 — Parameters
$A=-\tfrac{1}{2}$, $\omega=2$. Amplitude $= \tfrac{1}{2}$ (reflects over $x$-axis). Period $= \dfrac{2\pi}{2} = \pi$.
Step 2 — Key points (one cycle)
$x$ $0$ $\pi/4$ $\pi/2$ $3\pi/4$ $\pi$
$y$ $-\frac{1}{2}$ $0$ $\frac{1}{2}$ $0$ $-\frac{1}{2}$
Step 3 — Transformation
Start from $y=\cos x$: ① Horizontal compression by $\frac{1}{2}$ (period: $2\pi \to \pi$). ② Vertical compression to amplitude $\frac{1}{2}$, then reflect over $x$-axis.
Graph: y = −½ cos(2x) — two full cycles
Final Answer
Amplitude $= \tfrac{1}{2}$ (reflected),   Period $= \pi$ Domain: $(-\infty,\infty)$,   Range: $[-\tfrac{1}{2}, \tfrac{1}{2}]$

Problem 44

Graph using Transformations
Given
$$y = -4\sin\!\left(\frac{x}{8}\right)$$
Step 1 — Parameters
$A=-4$, $\omega=\tfrac{1}{8}$. Amplitude $= 4$ (reflected).   Period $= \dfrac{2\pi}{1/8} = 16\pi$.
Step 2 — Key points
$x$ $0$ $4\pi$ $8\pi$ $12\pi$ $16\pi$
$y$ $0$ $-4$ $0$ $4$ $0$
Step 3 — Transformation
Horizontal stretch by 8, vertical stretch by 4, and reflect over $x$-axis.
Graph: y = −4 sin(x/8) — two full cycles
Final Answer
Amplitude $= 4$ (reflected),   Period $= 16\pi \approx 50.27$ Domain: $(-\infty,\infty)$,   Range: $[-4, 4]$

Problem 49

Graph using Transformations — Vertical Shift
Given
$$y = -6\sin\!\left(\frac{\pi x}{3}\right) + 4$$
Step 1 — Parameters
$A=-6$, $\;\omega=\dfrac{\pi}{3}$, $\;B=4$. Amplitude $= 6$ (reflected).   Period $= \dfrac{2\pi}{\pi/3} = 6$.   Vertical shift up $4$.
Step 2 — Midline and range
Midline: $y = 4$. Range: $[4-6,\;4+6] = [-2, 10]$.
Step 3 — Key points (one cycle)
Quarter period $= \frac{6}{4} = \frac{3}{2}$.
$x$ $0$ $\frac{3}{2}$ $3$ $\frac{9}{2}$ $6$
$-6\sin(\pi x/3)$ $0$ $-6$ $0$ $6$ $0$
$y$ (after $+4$) $4$ $-2$ $4$ $10$ $4$
Graph: y = −6 sin(πx/3) + 4 — two full cycles
Final Answer
Amplitude $= 6$,   Period $= 6$,   Midline $y = 4$ Domain: $(-\infty,\infty)$,   Range: $[-2, 10]$

Problem 50

Graph using Transformations — Vertical Shift
Given
$$y = -3\cos\!\left(\frac{3\pi x}{4}\right) + 2$$
Step 1 — Parameters
$A=-3$, $\;\omega=\dfrac{3\pi}{4}$, $\;B=2$. Amplitude $= 3$ (reflected).   Period $= \dfrac{2\pi}{3\pi/4} = \dfrac{8}{3}$.   Vertical shift up $2$.
Step 2 — Midline and range
Midline: $y = 2$. Range: $[2-3,\;2+3] = [-1, 5]$.
Step 3 — Key points (one cycle)
Quarter period $= \frac{8/3}{4} = \frac{2}{3}$.
$x$ $0$ $\frac{2}{3}$ $\frac{4}{3}$ $2$ $\frac{8}{3}$
$-3\cos(\frac{3\pi x}{4})$ $-3$ $0$ $3$ $0$ $-3$
$y$ (after $+2$) $-1$ $2$ $5$ $2$ $-1$
Graph: y = −3 cos(3πx/4) + 2 — two full cycles
Final Answer
Amplitude $= 3$,   Period $= \tfrac{8}{3} \approx 2.667$,   Midline $y = 2$ Domain: $(-\infty,\infty)$,   Range: $[-1, 5]$
