★ Key Notes for this Section
Definition — Composite Function
$(f \circ g)(x) = f(g(x))$ The output of $g$ becomes the input of $f$. Domain of $f \circ g$: all $x$ in Dom($g$) such that $g(x)$ is in Dom($f$).
Procedure — Finding f ∘ g
1. Replace every $x$ in $f(x)$ with $g(x)$ and simplify. 2. Find domain restrictions: • Exclude values not in Dom($g$). • Exclude values where $g(x)$ is not in Dom($f$). 3. Note: $f \circ g \neq g \circ f$ in general (composition is not commutative).

Problem 17

Composite Function Values
Given
$f(x) = 2x+1$ and $g(x) = 3x$. Find (a) $(f \circ g)(4)$, (b) $(g \circ f)(2)$, (c) $(f \circ f)(1)$, (d) $(g \circ g)(0)$.
Compute each composite value
(a) $(f \circ g)(4) = f(g(4)) = f(12) = 2(12)+1 = 25$ (b) $(g \circ f)(2) = g(f(2)) = g(5) = 3(5) = 15$ (c) $(f \circ f)(1) = f(f(1)) = f(3) = 2(3)+1 = 7$ (d) $(g \circ g)(0) = g(g(0)) = g(0) = 0$
Exam Answer
Compute each composite value
(a) $(f \circ g)(4) = f(g(4)) = f(12) = 2(12)+1 = 25$ (b) $(g \circ f)(2) = g(f(2)) = g(5) = 3(5) = 15$ (c) $(f \circ f)(1) = f(f(1)) = f(3) = 2(3)+1 = 7$ (d) $(g \circ g)(0) = g(g(0)) = g(0) = 0$
Final Answer
(a) $25$, (b) $15$, (c) $7$, (d) $0$

$g(4) = \dfrac{3}{16+2} = \dfrac{3}{18} = \dfrac{1}{6}$ $f(2) = |2-2| = 0$ $f(1) = |1-2| = 1$ $g(0) = \dfrac{3}{0+2} = \dfrac{3}{2}$

Exam Answer
Find intermediate values of g and f
$g(4) = \dfrac{3}{16+2} = \dfrac{3}{18} = \dfrac{1}{6}$ $f(2) = |2-2| = 0$ $f(1) = |1-2| = 1$ $g(0) = \dfrac{3}{0+2} = \dfrac{3}{2}$
Compute each composite value
(a) $(f \circ g)(4) = f\!\left(\dfrac{1}{6}\right) = \left|\dfrac{1}{6}-2\right| = \dfrac{11}{6}$ (b) $(g \circ f)(2) = g(0) = \dfrac{3}{2}$ (c) $(f \circ f)(1) = f(1) = |1-2| = 1$ (d) $(g \circ g)(0) = g\!\left(\dfrac{3}{2}\right) = \dfrac{3}{\tfrac{9}{4}+2} = \dfrac{3}{\tfrac{17}{4}} = \dfrac{12}{17}$
