## Problem 17

**Type:** Composite Function Values  

### Given

> f(x) = 2x + 1; g(x) = 3x. Find (a) (f o g)(4), (b) (g o f)(2), (c) (f o f)(1), (d) (g o g)(0).

### Compute each composition value

(a) (f o g)(4) = f(g(4)) = f(12) = 25
(b) (g o f)(2) = g(f(2)) = g(5) = 15
(c) (f o f)(1) = f(f(1)) = f(3) = 7
(d) (g o g)(0) = g(g(0)) = g(0) = 0

### Final Answer

> (a) 25, (b) 15, (c) 7, (d) 0
